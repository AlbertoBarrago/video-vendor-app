# *POC* Video vendor app for Telegram

This is a Telegram mini app that allows users to purchase and download videos from a vendor.

## Features

- Purchase and download videos from a vendor
- View and purchase videos in a catalog
- Download videos to your device
- View and manage your purchases
- View and manage your downloads

## Installation

1. Clone the repository
 - Create the `.env` file for `BOT_TOKEN` and `MINI_APP_URL`
2. Install dependencies
3. Run the app `bun run dev`

## Usage

1. Open the app
2. Purchase and download videos from the vendor
3. View and manage your purchases and downloads
